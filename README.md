# Unit-Converter_-frontend-backend-

Unit Converter is a simple web application that allows users to convert values between different units of measurement.

This project is built using a **frontend + backend architecture** with communication through an API.

- **Frontend:** Built with **React** and styled using **Tailwind CSS**
- **Backend:** Built with **Express.js**

The frontend handles the user interface and user interactions, while the backend processes the conversion logic and sends the results back through the API.

This project was inspired by the **Unit Converter project idea from roadmap.sh**.
